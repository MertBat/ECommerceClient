About This Project;

Frontend: On the frontend, I leveraged powerful libraries such as Angular 15, Bootstrap, Alertif, Ngx-File-Drop, Ngx-Spinner, Ngx-Toastr, RxJS, and Angular MaterialUI. These tools facilitated the development of a user-friendly interface, enabling seamless interaction for customers. Additionally, external services like AWS S3, Azure Storage Blobs, and Facebook Authentication integrations with Google were provided to enhance the overall experience.

Backend: In the backend, I implemented .Net Onion architecture and CQRS pattern for order and product management, integrating an advanced authorization system. PostgreSQL database served as the foundation, with Entity Framework and Identity Framework managing database operations efficiently. Tools like MediatR, FluentValidation, JWTBearer, Serilog, and SignalR were instrumental in enhancing code quality.
